Web SQL Database

The database in HTML5 is still a problem currently.
The WebSQL Database is supported by Safari, Chrome, and Opera. But Firefox and IE is not support this.
The another one called IndexedDB will be supported by Firefox and Chrome in the future. (There's no browser support IndexedDB currently)

An important news is that W3C has announced that they will no longer maintain WebSQL in the future, but we can not sure wheather Safari and IE will support IndexedDB or not. Currently, you can only use WebSQL for offline database. But this database is not supported by Firefox and IE.
